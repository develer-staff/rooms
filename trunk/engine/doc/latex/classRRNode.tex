\subsection{Detailed Description}
The \hyperlink{classRRNode}{RRNode} class is used to navigate through xml tree. The class navigates through xml nodes and gets attributes from them. Example: 
\begin{DoxyCode}
 RRNode *node(validXmlElement);
 node->gotoElement("my_element");
 for (node->gotoChild("my_child"); !node->isNull(); node->gotoNext())
      //do something
\end{DoxyCode}
